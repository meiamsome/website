Checkbox
For adding a checkbox element to your project, first you must create a div with the class siimple-checkbox: 
<div class="siimple-checkbox"></div>
Inside it, create an input with the attribute type="checkbox". Also, you must add an id attribute that will be used in the last step. 
<div class="siimple-checkbox">
  <input type="checkbox" id="myCheckbox">
</div>
Finally, create a label element bound to the previous input element. 
<div class="siimple-checkbox">
  <input type="checkbox" id="myCheckbox">
  <label for="myCheckbox"></label>
</div>
Final result: 
<label class="siimple-label">Your checkbox:</label>
<div class="siimple-checkbox">
  <input type="checkbox" value="1" id="myCheckbox" name="">
  <label for="myCheckbox"></label>
</div>
Input
Add the siimple-input class to your text-based input fields (text, password and email types). 
<label class="siimple-label">Your email: </label>
<input type="text" class="siimple-input" placeholder="you@company.com">
You can set a full width input field by adding a siimple-input--fluid class. 

<label class="siimple-label">Your email: </label><br>
<input type="text" class="siimple-input siimple-input--fluid">
Label
Style your form label element adding a siimple-label class. 
<label class="siimple-label">Your name: </label>
<input type="text" class="siimple-input" placeholder="Your name">
Select
You can style your select elements adding a siimple-select class. 
<label class="siimple-label">Select an option: </label>
<select class="siimple-select">
  <option selected>Option 1</option>
  <option>Option 2</option>
  <option>Option 3</option>
</select>
Add siimple-select--fluid to set a fluid select size. 

<label class="siimple-label">Select an option: </label><br>
<select class="siimple-select siimple-select--fluid">
  <option selected>Option 1</option>
  <option>Option 2</option>
  <option>Option 3</option>
</select>

Textarea
Add the siimple-textarea class to your textarea element. You can set a fluid textarea by adding a siimple-textarea--fluid class. 

<label class="siimple-label">Your message: </label><br>
<textarea class="siimple-textarea siimple-textarea--fluid" rows="5">
  Type here your message
</textarea>
