Form
To create a form, the first thing you have to do is to create the div that will contain it and then add the siimple-form class: 
<div class="siimple-form"></div>
Next thing to do is to give a title and a detail to the form: 
<div class="siimple-form">
  <div class="siimple-form-title">Send a comment</div>
  <div class="siimple-form-detail">Fill the form to contact us.</div>
</div>
This is the look it will have once we add some form elements to it (don't worry, they will all be explained later):
Send a comment
Fill the form to contact us.
Your name
Send comment
  <div class="siimple-form">
    <div class="siimple-form-title">Send a comment</div>
    <div class="siimple-form-detail">Fill the form to contact us.</div>
    <div class="siimple-form-field">
       <div class="siimple-form-field-label">Your name</div>
       <input type="text" class="siimple-input siimple-input--fluid" placeholder="Your name">
    </div>
    <div class="siimple-form-field">
      <div class="siimple-btn siimple-btn--blue">Send comment</div>
    </div>
  </div>
Field
All the different sections of the form should be contained in fields. This way, every part of your form will be easily identifiable. Just create a div with the class siimple-form-field: 
<div class="siimple-form-field"></div>
Separate the different sections depending on their purpose with fields and add labels (using the class siimple-form-label) and helpers (using the class siimple-form-helper) to them. You'll end up with a clear and intuitive form that will provide a great user experience:
Your email
This field can't be empty
Your comment
This field can't be empty
  <div class="siimple-form-field">
     <div class="siimple-form-field-label">Your email</div>
     <input type="text" class="siimple-input siimple-input--fluid" placeholder="you@company.com">
     <div class="siimple-form-field-helper">This field can't be empty</div>
  </div>
  <div class="siimple-form-field">
     <div class="siimple-form-field-label">Your comment</div>
     <textarea class="siimple-textarea siimple-textarea--fluid"></textarea>
     <div class="siimple-form-field-helper">This field can't be empty</div>
  </div>
Checkbox
For adding a checkbox element to your project, first you must create a div with the class siimple-checkbox: 
<div class="siimple-checkbox"></div>
Inside it, create an input with the attribute type="checkbox". Also, you must add an id attribute that will be used in the last step. 
<div class="siimple-checkbox">
  <input type="checkbox" id="myCheckbox">
</div>
Finally, create a label element bound to the previous input element. 
<div class="siimple-checkbox">
  <input type="checkbox" id="myCheckbox">
  <label for="myCheckbox"></label>
</div>
Final result: 
<label class="siimple-label">Your checkbox:</label>
<div class="siimple-checkbox">
  <input type="checkbox" id="myCheckbox" checked>
  <label for="myCheckbox"></label>
</div>
Input
Add the siimple-input class to your text-based input fields (text, password and email types). 
<label class="siimple-label">Your email: </label>
<input type="text" class="siimple-input" placeholder="you@company.com">
You can set a full width input field by adding a siimple-input--fluid class. 

<label class="siimple-label">Your email: </label><br>
<input type="text" class="siimple-input siimple-input--fluid">
Label
Style your form label element adding a siimple-label class. 
<label class="siimple-label">Your name: </label>
<input type="text" class="siimple-input" placeholder="Your name">
Select
You can style your select elements adding a siimple-select class. 
<label class="siimple-label">Select an option: </label>
<select class="siimple-select">
  <option selected>Option 1</option>
  <option>Option 2</option>
  <option>Option 3</option>
</select>
Add siimple-select--fluid to set a fluid select size. 

<label class="siimple-label">Select an option: </label><br>
<select class="siimple-select siimple-select--fluid">
  <option selected>Option 1</option>
  <option>Option 2</option>
  <option>Option 3</option>
</select>
Switch
You can style a checkbox like a switch. First create a div element with a siimple-switch class: 
<div class="siimple-switch"></div>
Then add a input element inside your new switch div. You must also assign an id to the input element. 
<div class="siimple-switch">
  <input type="text" id="mySwitch">
</div>
Finally, add a label bound to the previously input and an empty div element. 
<div class="siimple-switch">
  <input type="text" id="mySwitch">
  <label for="mySwitch"></label>
  <div></div>
</div>
Final result: 
<label class="siimple-label">Your switch:
<div class="siimple-switch">
  <input type="checkbox" id="mySwitch" checked>
  <label for="mySwitch"></label>
  <div></div>
</div>
Textarea
Add the siimple-textarea class to your textarea element. You can set a fluid textarea by adding a siimple-textarea--fluid class. 

<label class="siimple-label">Your message: </label><br>
<textarea class="siimple-textarea siimple-textarea--fluid" rows="5">
  Type here your message
</textarea>
