Typography
Blockquote

Use siimple-blockquote class to style your blockquotes:

Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. 
<blockquote class="siimple-blockquote">
  Lorem ipsum dolor sit amet, consectetur...
</blockquote>
Code

You can use the siimple-code class adding a piece of code inline.

This is an inline code block. 
This is an <code class="siimple-code">inline</code> code block.

For multiline code blocks, use the siimple-pre class with the pre tag.

This is a full code block
<pre class="siimple-pre">This is a full code block</pre>

Link
Style your links adding the siimple-link class.
This is a text with a link element. 
This is a text with a <a href="#" class="siimple-link">link</a> element.
Paragraph

You can create a paragraph by adding the siimple-p or siimple-paragraph class to your div or p element.

Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat. 

<p class="siimple-p">
  Lorem ipsum dolor sit amet, consectetur adipiscing...
</p>
Small
Add siimple-small class to style your small tags.
This is a text with a small text content. 
This is a text with <small class="siimple-small">a small text</small> content.
Tag
Add the base siimple-tag class to a span or div element to style text as a tag. Then, add siimple-tag--[COLOR] to set the tag color.
Navy tag Green tag Teal tag Blue tag Purple tag Pink tag Red tag Orange tag Yellow tag Grey tag 
<span class="siimple-tag siimple-tag--navy">Navy tag</span>
<span class="siimple-tag siimple-tag--green">Green tag</span>
<span class="siimple-tag siimple-tag--teal">Teal tag</span>
<span class="siimple-tag siimple-tag--blue">Blue tag</span>
<span class="siimple-tag siimple-tag--purple">Purple tag</span>
<span class="siimple-tag siimple-tag--pink">Pink tag</span>
<span class="siimple-tag siimple-tag--red">Red tag</span>
<span class="siimple-tag siimple-tag--orange">Orange tag</span>
<span class="siimple-tag siimple-tag--yellow">Yellow tag</span>
<span class="siimple-tag siimple-tag--grey">Grey tag</span>
