Box
As its name suggests, it's just a simple box. It can contain up to three different elements: a title, a subtitle and a detail. To create a basic box, add to a div the class siimple-box: 
<div class="siimple-box"></div>
To use any of the three elements that box posesses just create a div for each one and apply the corresponding class: siimple-box-title, siimple-box-subtitle and siimple-box-detail:
Box title
This is the box subtitle
This is the box detail text
<div class="siimple-box">
  <div class="siimple-box-title">Box title</div>
  <div class="siimple-box-subtitle">This is the box subtitle</div>
  <div class="siimple-box-detail">This is the box detail text</div>
</div>
Don't forget that you can change the color of the box to any of the ones included in the colors section. To do it, just add the class siimple-box--COLOR to the box you've already created:
Box title
This is the box subtitle
This is the box detail text
<div class="siimple-box siimple-box--pink">
  <div class="siimple-box-title">Box title</div>
  <div class="siimple-box-subtitle">This is the box subtitle</div>
  <div class="siimple-box-detail">This is the box detail text</div>
</div>
Breadcrumb
The breadcrumb is an easy-to-use element to keep track of the path the user has navigated. To use it, first create a div to contain all the different breadcrumbs. Its class has to be siimple-breadcrumb: 
<div class="siimple-breadcrumb"></div>
Once you have the container of the breadcrumbs, just create as many as you want. A breadcrumb is just a div with the class siimple-breadcrumb-crumb added to it:
Home
A set of links
More links
  <div class="siimple-breadcrumb">
    <div class="siimple-breadcrumb-crumb">Home</div>
    <div class="siimple-breadcrumb-crumb">A set of links</div>
    <div class="siimple-breadcrumb-crumb">More links</div>
  </div>
Content
The content element is a tool that places the information it contains in the center of the window in every moment. Add the class siimple-content to your container div to use it: 
<div class="siimple-content"></div>
Remember that you can specify the size your content has using one of the three size tags siimple layout has: small, large and fluid. To apply them add to your content the corresponding class of the size desired: siimple-content--SIZE: 
<div class="siimple-size siimple-size--large"></div>
Footer
To create a footer, simply add the class siimple-footer to a div:
Made with love by siimple 
<div class="siimple-footer" align="center">
  Made with love by siimple
</div<
Remember that you can also choose the size and color of the footer. Apply the color you want from colors to the footer using siimple-footer--COLOR and the size using siimple-footer--SIZE:
Made with love by siimple 
  <div class="siimple-footer siimple-footer--teal siimple-footer--fluid" align="center">
    Made with love by siimple
  </div>
Jumbotron
The jumbotron is a pretty simple element. The only element it can contain is a title. To create a jumbotron just add the class siimple-jumbotron to a div, and then give it a title by putting inside a div with the class siimple-jumbotron-title:
Welcome page
<div class="siimple-jumbotron">
  <div class="siimple-jumbotron-title">Welcome page</div>
</div<
Customize your jumbotron by choosing a color from colors and adding by applying the class siimple-jumbotron--COLOR. The size is also customizable, just apply the class siimple-jumbotron--SIZE to the jumbotron and you're ready to go:
Welcome page
  <div class="siimple-jumbotron siimple-jumbotron--orange siimple-jumbotron--fluid">
    <div class="siimple-jumbotron-title">Welcome page</div>
  </div>
Navbar
To create a basic navbar, the first thing to do is to create a div with the class siimple-navbar. You should give it a title: add inside a div and apply the class siimple-navbar-title to it. This will be the result:
My blog name
  <div class="siimple-navbar">
    <div class="siimple-navbar-title">My blog name</div>
  </div>
A navbar can contain two groups: one on the right and one on the left. This is great to separate the title of your app from the navigation links. You can create the left group inside your navbar by adding a div with the class siimple-layout--left, and the same for the right group with siimple-layout--right. Let's add two links to it using the class siimple-link and see how it looks (we put the title in the left and the navigation in the right):
My blog name
Link 1 Link 2 
<div class="siimple-navbar">
  <div class="siimple-layout--left">
    <a class="siimple-navbar-title">My blog name</a>
  </div>
  <div class="siimple-layout--right">
    <a class="siimple-navbar-link">Link 1</a>
    <a class="siimple-navbar-link">Link 2</a>
  </div>
</div>
Lastly, remember that you can choose the color and size of the navbar element. Choose the color you want from colors and apply it to the element using the class siimple-navbar--COLOR. Same for the size: apply the size you want by using the class siimple-navbar--SIZE. This is the customized navbar:
My blog name
Link 1 Link 2 
<div class="siimple-navbar siimple-navbar--yellow siimple-navbar--fluid">
  <div class="siimple-layout--left">
    <a class="siimple-navbar-title">My blog name</a>
  </div>
  <div class="siimple-layout--right">
    <a class="siimple-navbar-link">Link 1</a>
    <a class="siimple-navbar-link">Link 2</a>
  </div>
</div>
